概述
算例1实现了一个稠密矩阵 LDL^T 求解器，用于求解三对角对称正定矩阵方程组。该程序首先生成一个三对角矩阵的 JSON 文件，然后从 JSON 文件加载数据并调用 LDL^T 求解器进行求解。

功能
生成三对角对称正定矩阵（K[i,i]=2, K[i,i±1]=-1）
使用纯手动实现的 LDL^T 分解与求解
计算残差范数与精确解误差
支持从 JSON 文件输入方程组

依赖
Python 3.8+
NumPy

运行方法
方式一：直接运行（生成并求解 n=5 的算例）
bash
python 算例1.py
方式二：修改矩阵规模后运行
编辑 算例1.py末尾的 main部分，修改 generate_example1_json的 n参数：

python
# 例如求解 n=100 的算例
generate_example1_json("example1.json", n=100)

然后运行：
bash
python 算例1.py

输出说明
程序运行后会输出以下信息：
矩阵 K、右端项 R、精确解 a_exact
L 矩阵和 D 向量（LDL^T 分解结果）
数值解 a
残差范数 ||R - K·a||
与精确解的最大误差和 L2 误差

文件说明
example1.json：生成的算例1输入文件（包含矩阵 K、右端项 R、精确解 a_exact）
算例1.py：主程序文件
