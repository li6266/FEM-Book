概述
算例4实现了一个大规模二维 Poisson 方程有限元求解器。使用 Q4（双线性四边形）单元对单位正方形区域进行网格划分，采用 2×2 Gauss 积分计算单元刚度矩阵，并调用 Intel MKL PARDISO 求解器进行高效求解。

功能
生成 Q4 四边形网格（支持 50×50、100×100、200×200 等规模）
使用 2×2 Gauss 积分计算单元刚度矩阵和载荷向量
组装稀疏总体刚度矩阵（COO → CSR 格式）
应用 Dirichlet 边界条件（u=0 on ∂Ω）
调用 Intel MKL PARDISO 求解器求解
计算最大误差和 L2 相对误差
绘制数值解云图、误差云图和三维曲面图

依赖
Python 3.8+
NumPy
SciPy
Matplotlib
pypardiso（Intel MKL PARDISO Python 接口）

安装依赖
bash
pip install numpy scipy matplotlib pypardiso
运行方法
直接运行（测试 50×50、100×100、200×200 三种网格）
bash
python 算例4.py
修改网格规模
编辑 算例4.py末尾的 grid_sizes列表：

python
# 例如只测试 100×100 和 200×200
grid_sizes = [100, 200]
输出说明

程序运行后会输出每个网格规模的以下信息：
单元类型、节点数、单元数、未知自由度数
总体矩阵非零元个数
装配时间、边界条件处理时间、求解时间、总时间
相对残差
节点最大误差和离散 L2 相对误差
同时会弹出以下图形窗口：
数值解云图 + 三维曲面图（组合图）
误差分布云图
三维曲面图

文件说明
算例4.py：主程序文件
运行时会在当前目录生成 PNG 图片文件

注意事项
200×200 网格约需 8-10 秒运行时间，请耐心等待
确保已安装 pypardiso，否则会报 ImportError
绘图窗口需要手动关闭才能继续下一个网格的计算
