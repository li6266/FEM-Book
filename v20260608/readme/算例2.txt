概述
算例2用于检测非正定矩阵。程序构造一个非正定矩阵 K = [[1, 2], [2, 1]]，尝试对其进行 LDL^T 分解，验证分解阶段的非正主元检测机制。

功能
生成非正定矩阵的 JSON 文件
调用 LDL^T 分解器进行分解
检测 D[j] ≤ 0 的情况并输出错误提示
验证求解器的鲁棒性

依赖
Python 3.8+
NumPy

运行方法
直接运行
bash
python 算例2.py
预期输出

程序运行后会输出以下信息：
纯文本
--- 开始求解算例2 ---
测试矩阵 K:
 [[1 2]
 [2 1]]
右端向量 R: [1, 1]
 计算中断：矩阵非正定：D[0] = -3.0

文件说明
example2.json：生成的算例2输入文件
算例2.py：主程序文件
