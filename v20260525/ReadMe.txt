程序简介
这是一个基于Python的有限元分析程序，专门用于分析一维和二维桁架结构。程序实现了完整的有限元分析流程，包括前处理、单元分析、刚度矩阵组装、方程求解和后处理等模块。
功能特性
1. 支持的结构类型
一维杆结构：简单的轴向拉伸/压缩问题
二维桁架结构：平面桁架结构分析
2. 核心功能模块
模块1：前处理​ - 从JSON文件读取模型数据
模块2：对号矩阵和直接组装​ - 生成LM矩阵和组装总体刚度矩阵
模块3：方程求解​ - 施加边界条件并求解（支持三种方法）
模块4：后处理​ - 计算单元应力和轴力
3. 边界条件处理方法
缩减法（必须实现）
罚函数法（可选实现）
修改法（可选实现）

输入文件格式
程序使用JSON格式的输入文件，包含以下主要部分：
基本信息
{
    "Title": "问题标题",
    "nsd": 空间维度,
    "ndof": 每个节点自由度数,
    "nnp": 节点总数,
    "nel": 单元总数,
    "nen": 每个单元节点数
}
节点坐标
"X": [
    [x1, y1],  // 节点1坐标
    [x2, y2],  // 节点2坐标
    // ...
]
单元连接
"IEN": [
    [1, 2],  // 单元1连接节点1和节点2
    [2, 3],  // 单元2连接节点2和节点3
    // ...
]
材料属性
"E": [E1, E2, ...],  // 弹性模量数组
"A": [A1, A2, ...]   // 横截面积数组
边界条件
"displacement_bc": [
    {
        "node": 节点编号,
        "dofs": [自由度列表],
        "values": [约束值列表]
    }
]
节点载荷
"nodal_loads": [
    {
        "node": 节点编号,
        "forces": [力向量]
    }
]
程序结构
主要类：TrussFEA
__init__(): 初始化有限元程序
read_input(): 前处理模块
compute_element_stiffness(): 单元分析模块
generate_LM(): 生成对号矩阵
assemble_global_stiffness(): 组装总体刚度矩阵
apply_boundary_conditions(): 方程求解模块
compute_stresses(): 后处理模块
run_analysis(): 主程序流程
辅助函数
create_example1_input_file(): 创建算例1输入文件
create_example2_input_file(): 创建算例2输入文件
analyze_example2(): 分析算例2并验证结果
main(): 主函数
常见问题
1. 程序无法运行
确保已安装Python 3.7+
确保已安装numpy库：pip install numpy
确保当前目录有写权限，程序会自动创建JSON文件
2. 结果不正确
检查输入文件格式是否正确
确认边界条件设置正确
检查材料属性单位是否一致
查看程序输出的调试信息，特别是刚度矩阵和载荷向量
3. 数值误差
程序使用双精度浮点数计算
对于刚度很大的结构，可能会出现数值问题
可以调整罚函数法的罚系数
扩展和修改
添加新算例
创建一个新的JSON输入文件
修改main()函数添加新的选择项
创建对应的分析函数
修改算法
可以在compute_element_stiffness()中修改单元刚度矩阵计算
可以在apply_boundary_conditions()中修改边界条件处理方法
可以在compute_stresses()中修改应力计算方法
验证和测试
程序包含自动验证功能：
算例1：验证位移和约束反力
算例2：验证节点位移、单元应力和刚度矩阵特性
所有验证都会显示"✓"表示正确或"✗"表示错误。
注意事项
程序会自动创建输入文件，无需手动准备
一维和二维问题使用不同的计算方法
程序输出大量中间信息，便于调试和理解
验证结果基于理论解，实际计算结果可能有微小误差
