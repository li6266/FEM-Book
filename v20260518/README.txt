程序简介
这是一个基于Python编写的三维杆单元（Truss Element）有限元分析程序。它可以计算给定几何和材料属性的杆单元的刚度矩阵、单元长度、方向余弦，并能根据给定的节点位移计算应变、应力和轴力。
基本属性计算：根据两个节点坐标计算单元长度和方向余弦。
刚度矩阵计算：计算单元在全局坐标系下的6x6刚度矩阵。
应力应变计算：根据给定的节点位移向量，计算杆单元的轴向应变、应力和轴力。
退化单元检查：自动检测节点重合的退化单元并报错。
核心函数接口
truss3d_element_stiffness(x1, x2, E, A)
功能：计算单元的基本属性和完整的6x6刚度矩阵。
输入：
x1, x2：节点坐标，形式为 [x, y, z]的NumPy数组。
E：弹性模量 (Pa)。
A：横截面积 (m²)。
输出：(L, direction_cosines, Ke)，即单元长度、方向余弦数组 [cx, cy, cz]和6x6刚度矩阵。
truss3d_element_stress(x1, x2, E, A, de)
功能：计算给定位移下的应变、应力和轴力。
输入：
x1, x2, E, A：同刚度矩阵函数。
de：单元节点位移向量，形式为 [u1, v1, w1, u2, v2, w2]。
输出：(epsilon, sigma, N)，即应变、应力 (Pa) 和轴力 (N)。
get_compact_stiffness_matrix(Ke, tolerance=1e-10)
功能：从完整的6x6刚度矩阵中去除所有元素均为零（在容差范围内）的行和列，生成紧凑的对称刚度矩阵。
输入：完整的6x6刚度矩阵 Ke和可选的容差 tolerance。
输出：(Ke_compact, active_dofs)，即紧凑刚度矩阵和对应的活动自由度索引列表。
如何使用
环境要求：确保已安装 Python 和 NumPy 库。
运行程序：直接运行 main2.py文件即可执行内置的两个测试算例。
自定义算例：要计算你自己的杆单元，请在 main2.py文件末尾的 if __name__ == “__main__”:代码块中，按照已有算例的格式修改以下参数：
node1, node2：两个节点的三维坐标。
E, A：材料的弹性模量和横截面积。
de：假定的节点位移向量。
内置测试算例
程序自带两个测试算例：
一维杆：沿X轴方向的杆，用于验证基本功能。
空间任意方向杆：节点坐标在三维空间任意方向的杆，用于验证程序的通用性。
每个算例的运行结果会依次打印出输入参数、基本属性、刚度矩阵（维度可变）以及应力应变计算结果。
输出说明
程序运行后会输出：
输入参数：确认输入的节点坐标、材料属性和位移。
基本属性：单元长度和方向余弦。
刚度矩阵：以紧凑形式打印出的对称刚度矩阵，并标注其维度和对称性检查结果。
应力应变结果：计算得出的应变值、应力值和轴力值。
错误提示：如果输入了重合的节点（退化单元），程序会捕获错误并给出明确提示。
注意事项
程序中所有物理量均采用国际单位制（SI）。
位移向量 de必须为包含6个元素的数组，分别对应两个节点在X, Y, Z三个方向的位移。
程序对“退化单元”（即两个节点重合，长度为0的单元）进行了检查，遇到此类情况会抛出 ValueError并终止计算。
